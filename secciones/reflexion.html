REFLEXION

Aprendí los fundamentos de la tecnología a través de la lectura de libros y recursos brindados por el docente. Comprendí conceptos clave sobre hardware, software y sistemas operativos de la actualidad.

Aprendí HTML a través de tutoriales y con la retroalimentacion del docente. Creé algunas páginas web simples para consolidar mi comprensión. Aprendí a estructurar contenido web usando etiquetas y a enlazar elementos.

Para aprender CSS, seguí tutoriales interactivos y realicé ejercicios de diseño web. Adquirí habilidades para estilizar mis páginas web, controlar el diseño y la presentación.

Aprendi JavaScript a través de tutoriales, desafíos de programación y proyectos prácticos. Aprendí conceptos básicos de programación como variables, condicionales y bucles, y comencé a escribir scripts interactivos.

Profundicé en JavaScript explorando conceptos avanzados como funciones, objetos y eventos. Resolví algunos ejemplos de programación más complejos enlazandolos a un proyecto de html.

Aprendí React mediante cursos en línea y la enseñanza del docente. Entendí cómo realizar la instalacion y crear proyectos basicos.

A lo largo de las seis semanas, desarrollé una comprensión sólida de la tecnología y las habilidades esenciales para la creación de sitios web interactivos. Aprendí cómo estructurar contenido con HTML, dar estilo con CSS y crear funcionalidades dinámicas utilizando JavaScript.
